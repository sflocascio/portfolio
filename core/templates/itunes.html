Song Findr

A Front-End web application for browsing and sampling content via the itunes API



Overview

Itunes has over 26 million songs on it’s webpage. Using itunes publicly available api, I designed an easy to use Front-End application that allows a user to search by any amount of text and retrieve any songs that match the search. Clicking on any search results plays a sample audio clip of the song.

Process

{% comment %}
    {% endcomment %}
    • Use Jquery to make an ajax call to the api, retrieve search results from user input

    • Render each song name, band name, thumbnail photo and song sample into one element

    • Display search results on the web page

    • Access and play each item's audio when it's clicked

This exploration into the itunes API laid the groundwork for a couple other music based projects I’ll be working on in the future.
