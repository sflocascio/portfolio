Song Findr

A front end web app for searching and sampling content via the itunes API.



Overview

Itunes has over 26 million songs on it’s webpage. Using itunes publicly available api, I designed a simple front end application that allows a user to search by any amount of text and retrieve any songs that match the search. Clicking on any search results plays a sample audio clip of the song.

Process

{% comment %}
    {% endcomment %}
    • Using Jquery to make an ajax call to the api, retrieving search results

    • Render each song name, band name, thumbnail photo and song sample into one element

    • Display search results

    • Access and play each items audio elements on click

Api’s give us the opportunity to access lots of public information. This exploration into the itunes Api laid the groundwork for a couple other music based projects I’ll be working on in the future.
