S a m   L o C a s c i o

Web Developer - Musician - Recording Engineer - Nature Enthusiast - Kayak Owner


I’m a natural creative who loves problem solving and the journey of building new things. My first exposure to coding happened sometime in the Myspace era when I began adding HTML and CSS to make my bands (and other bands) pages look better. It was this investigative process that led me to be intrigued by coding and technology. I love the outdoors, recording music, fishing, playing ping pong, and discovering new things. I recently received my Full stack Certification from Momentum Learning in Durham, NC and am currently working freelance on projects set to release in 2019.

Feel free to contact me if you want to discuss a project or opportunity.

My core aim in Front-End roles is to build efficient, interactive, and beautiful applications following DRY principles and W3 guidelines. In following with an agreed upon UI design, I approach projects with a mobile first mentality, making sure visual components are intentionally structured while bringing an emphasis on accessibility.

Tools: HTML5 , CSS3, javaScript, jQuery, VueJS, Bootstrap, Bulma

My goal and approach with UI and UX is to connect business goals with user needs through a process of testing and refinement that leads to a product that satisfies both sides of the relationship.

Tools: Sketch io, Figma, Illustrator, Photoshop

Special care is taken to ensure that the right tools are chosen so each project scales well as it grows. I bring an object oriented approach to my coding to assert the code base stays as concise as possible.

Tools: Python, Django, Amazon Webservices S3, PostgreSQL, Heroku, Django REST API

    {% comment %} Business Directory {% endcomment %}

    {% comment %} Business Directory {% endcomment %}

    Ideate solutions by creating wireframes and mockups. Gathering user and stakeholder feedback.


      3. Visual Design

    {% comment %} Business Directory {% endcomment %}

    Design a high-res prototype of the interface that reflects research and feedback from Card Creation phase.


      4. Front End Development


    {% comment %} Business Directory {% endcomment %}

    Prototype the agreed design in code using HTML, CSS, Jquery, and JavaScript



      5. User Testing

    {% comment %} Business Directory {% endcomment %}

    Perform research through series of usability and accessibility tests , evaluate design with multiple user scenarios.

    Redo the entire process and use new insights to improve on the product’s design and user experience.
