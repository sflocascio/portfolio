Wayfinder

An exploration in data visualazation with 360,000 data points



The Challenge

Given a set of 360,000 specific 4 digit elevation points, how does one make visual sense of it?

My approach was to write a Python script that could convert each data point into a specific value on the color scale (0-255). When rendered, all data points combined should make an image that visually reprsents the elevation differences in the data set.

{% comment %} {% endcomment %}

The Process

  • Clean Data- Take away commas, spaces, and any other characters so the data can be easily iterated over

  • Height and Depth - Identify max and min data points

  • Colorize - With my max and min numbers defined I could iterate over every data point and turn it into a color code using this formula: y - min_height / max_height - min_height * 255

  • Tools- Pillow is an excellent tool for python for manipulating images in python, this is what was used to create the image

  • Render With every data point set to a color code, Loop through the data set and create an image with each data point becoming a pixel represented by its unique color code

The Result

In less than a second I am able to render image into 360,000 pixels (or rather a 600x600 image). Sure enough this data set looks like a landscape, (yay!). It took me a couple tries to get it correctly, but as you can see, the darkest shades represent the lowest elevations in the data set and the brightest shades represent the highest.
